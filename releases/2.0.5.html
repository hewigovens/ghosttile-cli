• Detect apps with system extensions or driver entitlements and refuse to manage them, with a clear explanation that modifying them would break their drivers.
  • Warn before managing apps with TCC-protected entitlements (microphone, camera, contacts, calendars, location, photos, AppleEvents) — Hide-from-Dock requires re-signing, which revokes the original capability grant.
  • Compatibility warnings now show a confirmation dialog (Continue / Cancel) in the GUI and a y/N prompt in the CLI, instead of blocking with an error.
  • Preserve non-TCC entitlements (App Sandbox, application groups, keychain access, network, custom developer entitlements) during Hide-from-Dock. Previously they were silently stripped due to a codesign-extract bug that read from the wrong stream and returned empty.
  • App Management permission denials now show a guided alert pointing to System Settings → Privacy & Security → App Management, instead of a raw Operation not permitted error.
  • Fixed a launch crash for apps without an existing @executable_path/../Frameworks rpath (e.g. RME TotalMix FX). GhostTile now adds the rpath when needed so the injected helper resolves correctly.
